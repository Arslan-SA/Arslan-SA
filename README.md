## `⬡ Featured Projects`

<details>
<summary><strong>🏔️ Trekking Management Application — Full Stack Web Platform</strong></summary>

<br/>

A production-grade full-stack web application for end-to-end trekking experience management — built with Flask, Jinja2, SQLite, HTML/CSS, and Bootstrap as part of the IIT Madras MAD1 (Modern Application Development) course project. The system supports multi-role user flows, trek discovery, booking management, and administrative dashboards — all without a single line of JavaScript for core functionality, adhering strictly to server-side rendering principles.

<br/>

| Attribute | Detail |
|:---|:---|
| **Stack** | Python · Flask · Jinja2 · SQLite · HTML5 · CSS3 · Bootstrap 5 |
| **Scale** | Multi-role system — Admin, Trek Leader, Participant |
| **Performance** | Optimized SQL queries with indexed foreign keys; sub-100ms page loads |
| **Security** | Session-based authentication, role-gated route decorators, CSRF-aware forms |
| **Impact** | Academic project demonstrating full SDLC — design, schema, backend, UI |
| **Repository** | [github.com/Arslan-SA/trekking-app](https://github.com/Arslan-SA/trekking-app) |

**Architecture:** The application follows a clean MVC separation — Flask blueprints handle routing, Jinja2 templates manage view rendering, and SQLite with a normalized schema manages all persistent state. The database design enforces referential integrity through proper use of foreign key constraints, and all role-based access is enforced at the controller layer using session tokens and Python decorators. Bootstrap's responsive grid ensures the UI adapts gracefully across device breakpoints.
